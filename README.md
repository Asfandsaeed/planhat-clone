# Planhat Clone

Pixel-perfect clone of [planhat.com](https://www.planhat.com/) built with Next.js 16, Tailwind CSS v4, and shadcn/ui.

## Tech Stack

- **Framework:** Next.js 16 (App Router, React 19, TypeScript)
- **Styling:** Tailwind CSS v4
- **UI:** shadcn/ui

## Commands

- `npm run dev` — Start dev server
- `npm run build` — Production build

## Deployed
