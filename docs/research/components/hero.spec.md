# Hero Section Spec

## Visual Description
- Full-width branded hero with centered text
- Large serif headline, subdued subheading, muted description
- CTA button (black pill), trust bar with link
- "discover" text with arrow below

## Structure
```
┌──────────────────────────────────────┐
│         The Agentic Customer        │
│               Platform              │
│                                     │
│  Deploy AI with confidence across   │
│        the customer lifecycle        │
│                                     │
│  Planhat provides B2B enterprises  │
│  with forward-deployed software...  │
│                                     │
│      [ Request a Demo ]             │
│                                     │
│  Trusted by... ── Discover →        │
│              discover →             │
└──────────────────────────────────────┘
```

## Typography
- Headline: font-serif (Instrument Serif / Times Now SemiLight), 5xl→7xl→8xl, tracking-tight
- Subhead: text-lg→xl, text-muted-text
- Description: text-sm→base, text-muted-text/80
- CTA: text-sm, font-medium

## Spacing
- Section padding: pt-20 pb-24 → pt-28 pb-32
- Headline margin-top: mt-6
- Description margin-top: mt-4
- CTA margin-top: mt-10
- Trust bar margin-top: mt-16

## Responsive
- Mobile: centered stack, smaller font sizes
- Tablet: base sizes
- Desktop: max 7xl/8xl headline

## Background
- White (#FFFFFF / oklch(1 0 0))
- Hero image can be added as background
